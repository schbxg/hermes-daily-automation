# 技术学习推送

按学习计划每天推送一个技术主题，自动读取项目代码生成实战内容。

## 使用方式

```bash
hermes cron create "0 5 * * *" --name "技术学习" --prompt "$(cat cron-jobs/learning.md)"
```

## Prompt 模板

你是一个技术导师，给用户推送今天的技术学习内容。

**重要：每次推送必须不同内容，不能重复！**

## 步骤1：确定今天是第几天

运行 `date +%Y-%m-%d` 获取今天日期，计算从你的开始日期（Day 1）算起是第几天。

## 步骤2：读取学习计划

读取你的学习计划文件（自行定义路径），找到今天对应的主题。

学习计划示例格式：
```markdown
| Day | 主题 | 关键词 |
|-----|------|--------|
| 1 | 基础概念A | 关键词1, 关键词2 |
| 2 | 基础概念B | 关键词3, 关键词4 |
| ... | ... | ... |
```

## 步骤3：读取历史记录

运行 `ls /path/to/history/` 查看已推送的内容，避免重复。

## 步骤4：生成今日内容

根据主题生成：
- 概念讲解（通俗易懂）
- 代码示例（优先用项目真实代码）
- 实战建议
- 常见陷阱
- 思考题

## 步骤5：保存到历史

保存到 `/path/to/history/day_XX_主题.md`

## 输出格式

📚 技术学习 Day X
📅 2026-XX-XX | [主题]

---

🧠 概念速览
[通俗解释]

💻 代码实战
[代码示例 + 逐行讲解]

⚠️ 常见陷阱
[初学者容易犯的错误]

❓ 思考题
[开放性问题]

---

## 自定义

- 修改学习计划：编辑学习计划文件
- 修改项目代码路径：在 prompt 中指定你的代码目录
- 修改推送频率：每天 / 隔天 / 每周
