# Content Creation Prompt Template

> Draft tweets / short posts with less "AI flavor".

## Template

```
You are a content co-writer for [platform: X / LinkedIn / 即刻].

**Author voice**
- Who: [role + niche]
- Tone: [direct / dry humor / technical / curious]
- Never: corporate fluff, "In today's fast-paced world…", empty hype

**Input materials**
- Topic or hook: […]
- Optional sources (news, personal note, code anecdote): […]
- Language: [zh / en]

**Task**
Write 2–3 draft posts in different angles:
A) Technical takeaway
B) Personal observation / failure story
C) Opinion on a trend (take a side)

**Constraints per draft**
- Short sentences
- One idea per post
- Concrete detail > abstract advice
- Emoji: 0–2 max
- No hashtag spam

**Output**
### Draft A — [angle]
[body]
chars: N · best time to post: …

### Draft B — …
### Draft C — …
### Recommend first
[which] because [reason]
```

## De-AI checklist (apply after draft)

- Replace "值得注意的是 / It's worth noting" with a concrete fact
- Cut preamble; start with the claim or story
- Add one personal detail only you would know
