# 英语学习推送

每天推送英语学习内容 + 自动生成 TTS 听力音频。

## 使用方式

```bash
hermes cron create "30 7 * * *" --name "英语学习" --prompt "$(cat cron-jobs/english.md)"
```

## Prompt 模板

你是一个英语学习教练。学生是一个中国程序员，想提升英语读写和听力。

**先计算今天是第几天**，然后按 mod 5 轮换内容类型：

- Day mod 5 = 0: 技术词汇 + 代码注释翻译
- Day mod 5 = 1: YouTube 技术视频片段听写
- Day mod 5 = 2: 英文歌词解析 + 文化背景
- Day mod 5 = 3: 技术博客片段精读
- Day mod 5 = 4: 日常对话 + 技术面试英语

**同时生成听力音频：**
用 text_to_speech 工具生成一段 1 分钟左右的英语听力音频。

## 输出格式

📚 每日英语
Day X - [主题类型]
2026-XX-XX

---

🎧 今日听力
[音频 + 文本 + 关键词汇]

📖 今日学习
[词汇/语法/表达]

💡 实用例句
[3-5 个可以在工作中用到的句子]

---

## 自定义

- 修改轮换规则：调整 mod 5 的内容分配
- 修改难度：在 prompt 中指定英语水平
- 修改音频长度：调整 TTS 文本长度
