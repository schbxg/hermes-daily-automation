# Prompt 模板

> 可直接复制使用的 Prompt 模板，适用于各种 AI 场景

## 模板列表

| 模板 | 用途 | 适用模型 |
|------|------|----------|
| [技术学习](tech-learning.md) | 按计划学习新技术 | Claude/GPT/MIMO |
| [代码审查](code-review.md) | AI 辅助代码审查 | Claude/GPT |
| [面试准备](interview-prep.md) | 面试题生成+答案 | Claude/GPT/MIMO |
| [内容创作](content-creation.md) | 推文/文章生成 | Claude/GPT |
| [Debug 助手](debug-helper.md) | 定位和修复 Bug | Claude/GPT |

## 使用方式

1. 复制模板内容
2. 替换 `[占位符]` 为你的实际内容
3. 发送给 AI 模型
4. 根据输出调整模板

## 自定义

你可以根据自己的需求修改模板，添加更多细节或约束条件。
