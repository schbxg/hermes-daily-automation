# 每日思考题

每天推送一个能引发深度思考的问题，不同领域轮换。

## 使用方式

```bash
hermes cron create "0 8 * * *" --name "每日思考" --prompt "$(cat cron-jobs/thinking.md)"
```

## Prompt 模板

用中文提出一个能引发深度思考的问题。

**重要：每次问题不同，不能重复！**

## 步骤1：确定领域

计算今天是第几天，按 mod 7 轮换：
- Day 1: 人生哲学与存在主义
- Day 2: 创造力与创新思维
- Day 3: 决策与判断
- Day 4: 人际关系与社会
- Day 5: 自我认知与成长
- Day 6: 世界观与价值观
- Day 0: 科技与人性

## 步骤2：读取历史

读取最近 5 天的记录，确保主题不同。

## 步骤3：生成问题

问题要：
- 有深度，能触发反思
- 有多个思考角度
- 不是知识性问题，而是开放性问题

## 输出格式

🧠 每日一问
Day X - [领域]
2026-XX-XX

---

[一个深刻的问题]

---

💡 试着从这几个角度想一想：
- [角度1]
- [角度2]
- [角度3]

[一句富有哲理的结尾]

---

## 自定义

- 修改领域：添加/删除/替换领域
- 修改风格：学术 / 通俗 / 文艺
